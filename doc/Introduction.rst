============
MTBindingSim
============

.. image:: $(IMAGES)/logo
   :width: 5in
   :align: center

.. raw:: latex
   
   \vspace{1in} \par



MTBindingSim is a program to plot binding curves under various conditions, 
particularly those encountered in microtubule binding or other polymer binding 
situations.
