A program to plot binding curves under various conditions.

Current Release

Our current release is version 0.1, a beta released on April 24, 2013. To download and install BindingTutor, see the downloads section below. You may also read the BindingTutor user manual online.

Find a Bug?

BindingTutor is under active development. If something doesn't work right for you, or you'd like to see a new feature, you're encouraged to submit an issue.

Prerequisites

The MATLAB Compiler Runtime must be installed on your computer in order to run BindingTutor. You only need to install it once. Download the appropriate file, and double-click it to extract. Then, double-click the InstallForMacOSX application found in the newly created folder. Follow the instructions to install the MCR.

Download MATLAB Compiler Runtime

Get BindingTutor

Then, download BindingTutor. Double-click to open the disk image and drag the BindingTutor icon to your Applications folder.
